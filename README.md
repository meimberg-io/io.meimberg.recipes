# Recipe Menu App

A Next.js application to display recipes from Notion, styled like a restaurant menu.

## Setup

See [docs/requirements/SETUP.md](docs/requirements/SETUP.md) for detailed setup instructions.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env.local` from `env.example`:
   ```bash
   cp env.example .env.local
   ```

3. Add your Notion credentials to `.env.local`

4. Start development server:
   ```bash
   npm run dev
   ```

Visit http://localhost:3000

## Tech Stack

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Notion API
